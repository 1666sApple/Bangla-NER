About Our Service

This is web application that provides Name Entity Recognition (NER) and Part of Speech (POS) Tagging for text input for Bengali language.
This was developed by 1666sApple as a part of the Bangla-NER project.
This project uses the Bangla-NER model, which is a pre-trained xml-roberta-base model for Bengali language.
The model is trained on a dataset of Bengali text that contains about 7000 sentences and is able to accurately identify and classify named entities and parts of speech.
This model has achived POS Precision of about 93.70% and NER Precision of about 92.56%

Features

  • Named Entity Recognition: Identify and classify named entities in text for Bengali language
  • Part of Speech Tagging: Determine the grammatical parts of speech for each word.
