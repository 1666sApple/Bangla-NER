About Our Service

This is a web application that provides Name Entity Recognition (NER) and Part of Speech (POS) Tagging for text input in Bengali language. Developed by 1666sApple as part of the Bangla-NER project.

The project uses the Bangla-NER model, a pre-trained xml-roberta-base model for the Bengali language.

Trained on a dataset of Bengali text containing about 7000 sentences, it accurately identifies and classifies named entities and parts of speech.

The model has achieved a POS Precision of about 93% and NER Precision of about 92% on the test data.

Features

  • Named Entity Recognition: Identify and classify named entities in Bengali text.
  • Part of Speech Tagging: Determine the grammatical parts of speech for each word.
